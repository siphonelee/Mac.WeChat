 注册后将授权配置到IPADDemo.exe.config即可使用<br/>

关闭windows数据保护DEP（dep可能会导致不安全代码闪退）：
新机器只需要首次进入cmd，输入 bcdedit.exe/set {current} nx AlwaysOff，重启电脑<br/>
